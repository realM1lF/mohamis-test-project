# Technologie-Stack

## Haupttechnologien

- **Sprache**: Python 3.11
- **Framework**: FastAPI
- **Datenbank**: SQLite
- **Frontend**: React

## Tools

- Docker für Containerisierung
- GitHub für Versionskontrolle
- Docker Compose für Orchestration

## Besonderheiten

- Keine externen APIs ohne Absprache
- Alles muss in Docker laufen
- Testing ist Pflicht
